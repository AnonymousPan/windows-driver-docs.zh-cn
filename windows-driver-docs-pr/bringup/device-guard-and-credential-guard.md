# <a name="hypervisor-protected-code-integrity-hvci"></a>受虚拟机监控程序保护的代码完整性（要求 HVCI）

受虚拟机监控程序保护的代码完整性可以使用硬件技术和虚拟化将代码完整性（CI）决策功能与 Windows 操作系统的其余部分隔离开来。 使用基于虚拟化的安全性来隔离代码完整性时，内核内存可执行的唯一方式是通过代码完整性验证。
