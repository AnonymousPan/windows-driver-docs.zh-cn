# <a name="markdevicepower-rule-wdm"></a>MarkDevicePower 规则（wdm）


**MarkDevicePower**规则指定 irp \_ MJ \_ power With IRP \_ MN \_ 将 \_ SystemPowerState irp 的 power 设置为 "正在挂起"。

此规则仅适用于 FDO 和 FIDO 驱动程序。

**驱动程序模型： WDM**
