# <a name="initfreedevicecallback-rule-kmdf"></a>InitFreeDeviceCallback 规则（kmdf）


InitFreeDeviceCallback 规则指定，如果驱动程序在初始化新框架设备对象时遇到错误，则驱动程序必须调用[**WdfDeviceInitFree**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdeviceinitfree) ; 如果驱动程序从对[**WdfControlDeviceInitAllocate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfcontrol/nf-wdfcontrol-wdfcontroldeviceinitallocate)的调用接收到[**WDFDEVICE \_ INIT**](https://docs.microsoft.com/windows-hardware/drivers/wdf/wdfdevice_init)结构，则为。

**驱动程序模型： KMDF**
