# <a name="doublekesetevent-rule-storport"></a>DoubleKeSetEvent 规则（storport）


此规则验证是否未在同一事件对象上两次调用**KeSetEvent** 。 如果将相同的事件对象传递到例程，则驱动程序将无法满足规则。

**驱动程序模型： Storport**
