# <a name="doubleexfreepool-rule-storport"></a>DoubleExFreePool 规则（storport）


此规则验证驱动程序是否不会尝试释放相同的池内存块。

规则将跟踪首先传递到**ExFreePool**的内存指针。 如果再次传递同一指针，则驱动程序将无法满足规则。 如果驱动程序调用**RemoveHeadList**或**RemoveEntryList**，则规则通过。

**驱动程序模型： Storport**
