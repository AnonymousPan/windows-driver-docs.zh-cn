# <a name="c28728"></a>C28728


警告 C28728：已禁止的 API 用法

此警告表明，使用的函数已被禁止或弃用，并且具有更可靠和更安全的替换。

若要查找替换，请在 [Windows 驱动程序开发](../index.yml) 文档中搜索弃用的函数的名称。 文档列出了弃用函数的名称并建议替换。
