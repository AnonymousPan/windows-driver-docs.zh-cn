# <a name="init_deregisterinterrupt-rule-ndis"></a>Init \_ DeRegisterInterrupt 规则（ndis）


**Init \_ DeRegisterInterrupt**规则指定在 MPInitilize 期间至少调用[**NdisMRegisterInterruptEx**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nf-ndis-ndismregisterinterruptex)一次， [**NdisMDeregisterInterruptEx**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nf-ndis-ndismderegisterinterruptex)应在 MPHaltEx 中至少调用一次。

如果初始化过程中出现问题或在停止微型端口过程中出现问题，则在初始化期间注册中断通常会发生。

**驱动程序模型： NDIS**
