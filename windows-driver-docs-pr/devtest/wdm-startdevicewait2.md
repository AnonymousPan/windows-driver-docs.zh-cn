# <a name="startdevicewait2-rule-wdm"></a>StartDeviceWait2 规则（wdm）


**StartDeviceWait2**规则指定驱动程序不应在启动设备 IRP 的上下文中调用[**KeWaitForSingleObject**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-kewaitforsingleobject) 。

**驱动程序模型： WDM**
