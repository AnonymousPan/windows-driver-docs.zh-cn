# <a name="spduplex-rule-storport"></a>SpDuplex 规则（storport）


此规则验证此小型端口**是否处于全双工模式。** 根据 StorPort 微型端口型号构建的任何驱动程序都必须处于**全双工模式。** 仅应在将现有 SCSI 驱动程序移植到 StorPort 时使用**半双工**。

**驱动程序模型： Storport**
