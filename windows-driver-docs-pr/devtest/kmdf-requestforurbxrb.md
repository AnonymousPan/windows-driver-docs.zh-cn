# <a name="requestforurbxrb-rule-kmdf"></a>RequestForUrbXrb 规则（kmdf）


如果客户端驱动程序调用[**WdfUsbTargetDeviceCreateWithParameters**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfusb/nf-wdfusb-wdfusbtargetdevicecreatewithparameters) ，并 \_ \_ \_ \_ 在 WDF \_ USB \_ 设备 \_ 创建 \_ 配置结构（以使用适用于 Windows 8 的 USB 驱动程序堆栈的新功能）中指定客户端协定版本 USBD 客户端协定版本602，则在内部使用 URB 的 DDIs 将仅在以下任何前提条件适用时才使用*URB 上下文*：

-   请求参数的父对象树中包含 Wdf 设备。
-   请求通过 i/o 队列来表示。
-   请求在其父对象树中有另一个表示请求的 i/o 队列。

**驱动程序模型： KMDF**
