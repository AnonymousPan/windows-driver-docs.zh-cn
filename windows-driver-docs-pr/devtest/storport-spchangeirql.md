# <a name="spchangeirql-rule-storport"></a>SpChangeIrql 规则 (storport) 


此规则验证 StorPort 回调例程是否返回的 IRQL 级别与调用它们的级别相同。

适用于更改 Storport 小型输入点或回调例程内的 IRQL 级别的任何 DDI。

**驱动程序模型： Storport**
