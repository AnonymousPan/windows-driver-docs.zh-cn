# <a name="driverattributechanged-rule-kmdf"></a>DriverAttributeChanged 规则（kmdf）


**DriverAttributeChanged**规则指定驱动程序不得更改 KMDF 驱动程序的执行级别或同步作用域。

**驱动程序模型： KMDF**
