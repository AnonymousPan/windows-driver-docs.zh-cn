# <a name="queuedspinlockrelease-rule-storport"></a>QueuedSpinLockRelease 规则（storport）


此规则验证驱动程序在未首先通过**KeAcquireInStackQueuedSpinLock**获取锁定的情况下不会调用**KeReleaseInStackQueuedSpinLock** 。

**驱动程序模型： Storport**
