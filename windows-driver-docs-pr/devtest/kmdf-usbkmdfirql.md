# <a name="usbkmdfirql-rule-kmdf"></a>UsbKmdfIrql 规则（kmdf）


**UsbKmdfIrql**规则指定 KMDF 驱动程序不会在错误的 IRQL 级别调用 USB 特定的设备驱动程序接口（DDI）。

**驱动程序模型： KMDF**
