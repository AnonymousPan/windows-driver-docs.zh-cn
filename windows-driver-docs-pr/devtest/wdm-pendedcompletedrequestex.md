# <a name="pendedcompletedrequestex-rule-wdm"></a>PendedCompletedRequestEx 规则（wdm）


**PendedCompletedRequestEx**规则指定，驱动程序不应为挂起的 IRP 调用[**IoCompleteRequest**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iocompleterequest) 。

**驱动程序模型： WDM**
