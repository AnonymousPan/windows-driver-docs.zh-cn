# <a name="syncreqsend-rule-kmdf"></a>SyncReqSend 规则（kmdf）


**SyncReqSend**规则指定所有同步发送请求都是使用特定于同步的 KMDF 设备驱动程序接口方法来完成的，并且这些方法的超时值设置为非零值。

如果驱动程序在不设置有效超时的情况下调用*WDFxxxSendXXXSynchronously*方法，则如果硬件未及时响应，则线程可能会停止。

**驱动程序模型： KMDF**
