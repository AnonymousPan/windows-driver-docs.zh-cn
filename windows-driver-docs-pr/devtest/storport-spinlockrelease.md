# <a name="spinlockrelease-rule-storport"></a>SpinLockRelease 规则（storport）


此规则验证驱动程序不会尝试通过**KeReleaseSpinLock**释放锁定，无需先通过**KeAquireSpinlock**或**KeAcquireSpinLockRaiseToDpc**获取锁定。 当释放获取的旋转锁时，规则通过。

**驱动程序模型： Storport**
