# <a name="usbkmdfirql2-rule-kmdf"></a>UsbKmdfIrql2 规则（kmdf）


**UsbKmdfIrql2**规则指定 KMDF 驱动程序不应在错误的 IRQL 级别调用特定于 USB 的 DDIs。

**驱动程序模型： KMDF**
