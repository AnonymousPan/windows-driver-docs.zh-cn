# <a name="proximity-sensor-property"></a>邻近感应传感器属性

此属性是一个可选的枚举属性。

| 属性键 | 类型|Access (R/O，R/W)  |必需/可选 | 说明 |
| --- | --- | --- | --- | --- |
|DEVPKEY_Sensor_ProximityType|VT_UI4|R/O|可选|描述检测到的邻近类型。 它可以是 HumanProximity 或 ObjectProximity。 有关详细信息，请参阅 ProximityType 枚举。|

**标头：** Sensorsdef
