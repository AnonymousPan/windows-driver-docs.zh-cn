# <a name="hot-spot-authentication"></a>热点身份验证


应尽可能将数据流量卸载到 Wi-fi 热点。 Windows 连接管理器可以在 Wi-fi 网络可用时自动连接到该网络。 若要连接到 Wi-fi 网络，必须向 Windows 提供热点凭据。 可以通过在应用中实现自己的身份验证来提供热点凭据。
