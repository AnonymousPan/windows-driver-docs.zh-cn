# <a name="sim-pin-api"></a>SIM PIN API


通过使用 SIM 卡 PIN API，移动宽带应用可以支持与 SIM 卡 PIN 和 Pin 解锁密钥相关的操作 (PUK) 。

有关 SIM 卡 PIN API 的详细信息，请参阅 [移动宽带 Api 接口](/windows/desktop/mbn/mobile-broadband-networks-api-interfaces)。
