# <a name="bug-check-0xb-no_exception_handling_support"></a>Bug 检查0xB：\_处理\_支持时不\_异常


\_处理\_支持 bug 检查的 "不\_" 异常的值为 "0x0000000B"。

此 bug 检查很少出现。


## <a name="resolution"></a>分辨率
[ **！分析**](https://docs.microsoft.com/windows-hardware/drivers/debugger/-analyze)调试扩展显示有关 bug 检查的信息，可帮助确定根本原因。
