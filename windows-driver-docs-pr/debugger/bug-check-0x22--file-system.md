# <a name="bug-check-0x22-file_system"></a>Bug 检查0x22：文件 \_ 系统


文件 \_ 系统 bug 检查的值为0x00000022。

此 bug 检查很少出现。

> [!IMPORTANT]
> 本主题适用于程序员。 如果你是在使用计算机时收到蓝屏错误代码的客户，请参阅[排查蓝屏错误](https://www.windows.com/stopcode)。
