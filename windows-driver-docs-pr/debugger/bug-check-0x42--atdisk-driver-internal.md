# <a name="bug-check-0x42-atdisk_driver_internal"></a>Bug 检查0x42： ATDISK \_ 驱动程序 \_ 内部


ATDISK \_ 驱动程序 \_ 内部 bug 检查的值为0x00000042。

此 bug 检查很少出现。

> 本主题适用于程序员。 如果你是在使用计算机时收到蓝屏错误代码的客户，请参阅[排查蓝屏错误](https://www.windows.com/stopcode)。
