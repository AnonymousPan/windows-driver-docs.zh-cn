# <a name="bug-check-0x1000007e-system_thread_exception_not_handled_m"></a>Bug 检查0x1000007E：系统 \_ 线程 \_ 异常 \_ 未 \_ 处理 \_ M


系统 \_ 线程 \_ 异常 \_ 未 \_ 处理 \_ M Bug 检查的值为0x1000007E。 这表示系统线程生成了错误处理程序未捕获的异常。

Bug 检查0x1000007E 与[**bug 检查 0x7E**](bug-check-0x7e--system-thread-exception-not-handled.md)具有相同的含义和参数（系统 \_ 线程 \_ 异常 \_ 未 \_ 处理）。

> [!IMPORTANT]
> 本主题适用于程序员。 如果你是在使用计算机时收到蓝屏错误代码的客户，请参阅[排查蓝屏错误](https://www.windows.com/stopcode)。

## <a name="resolution"></a>解决方法
[**！分析**](-analyze.md)调试扩展显示有关 bug 检查的信息，可帮助确定根本原因。
