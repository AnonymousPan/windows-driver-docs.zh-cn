# <a name="usbkdusbpdoxls"></a>!usbkd.usbpdoxls


**！ Usbkd. usbpdoxls**命令显示有关 USB 集线器驱动程序创建的所有物理设备对象（PDOs）的信息。 对于每个 PDO，此命令显示一行以逗号分隔的值。

```dbgcmd
!usbkd.usbpdoxls
```
