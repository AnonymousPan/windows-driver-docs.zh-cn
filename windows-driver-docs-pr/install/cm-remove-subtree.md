# <a name="cm_remove_subtree"></a>CM_Remove_SubTree

此函数保留供系统使用。

此功能已过时，Windows 2000 和更高版本的 Windows 不再支持此功能。 改用 [**CM_Query_And_Remove_Subtree**](/windows/desktop/api/cfgmgr32/nf-cfgmgr32-cm_query_and_remove_subtreew) 。
