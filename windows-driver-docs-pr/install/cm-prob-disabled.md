# <a name="code-22---cm_prob_disabled"></a>代码 22-CM_PROB_DISABLED

此设备管理器错误消息指示设备已禁用。

## <a name="error-code"></a>错误代码

22

### <a name="display-message"></a>显示消息

"此设备已禁用。 （代码22） "

### <a name="recommended-resolution"></a>建议的解决方法

设备被禁用，因为用户使用设备管理器禁用了它。 选择 "**启用**设备"，这将启用设备。
