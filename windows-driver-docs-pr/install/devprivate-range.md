# <a name="devprivate_range"></a>DEVPRIVATE_RANGE


**注意**   此结构仅供内部使用。

 

DEVPRIVATE_RANGE 结构指定资源需求列表，该列表描述设备实例的专用于设备的专用资源使用情况。 有关资源需求列表的详细信息，请参阅 [硬件资源](../kernel/hardware-resources.md)。
