# <a name="ksproperty_audio_loudness"></a>KSPROPERTY \_ 音频 \_ 响度


KSPROPERTY \_ AUDIO \_ 响度属性指定是否为响度节点 ([**KSNODETYPE \_ 响度**](ksnodetype-loudness.md)) 中的通道启用或禁用响度 (总体动态范围和低音提升) 。
