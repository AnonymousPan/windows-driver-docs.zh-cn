| [设备控制台 (devcon) ](/samples/microsoft/windows-driver-samples/device-console-devcon-tool/) | 一个命令行工具，用于显示有关设备的详细信息，并使你能够从命令行搜索和操作设备。 在本地计算机上，Devcon 启用、禁用、安装、配置和删除设备。 它还显示有关本地和远程计算机上的设备的详细信息。 |
