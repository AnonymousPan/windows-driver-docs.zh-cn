# <a name="trusted-certificates"></a>受信任的证书


Windows 使用受信任的证书验证在初始设置时从移动网络运营商的网页或从强制网络门户传递的设置文件上的数字签名。 这允许用户在安装移动宽带应用之前购买服务。
