# <a name="srb_surprise_removal"></a>SRB \_ 意外 \_ 删除

类驱动程序在从计算机上意外删除了某个设备 (意外删除) 时发送此请求。

有关详细信息，请参阅 [删除设备](../kernel/removing-a-device-in-a-function-driver.md)。
