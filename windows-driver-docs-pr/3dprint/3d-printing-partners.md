Microsoft 已与3D 硬件合作伙伴合作，使使用 Windows 10 的任何人都可以访问端到端的3D 生产。 有关合作伙伴和设备的最新信息，请参阅[3d 打印合作伙伴](https://www.microsoft.com/en-us/3d-print/printing-partners)。
