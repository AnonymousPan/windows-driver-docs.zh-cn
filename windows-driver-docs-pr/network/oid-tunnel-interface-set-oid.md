# <a name="oid_tunnel_interface_set_oid"></a>OID_TUNNEL_INTERFACE_SET_OID

OID_TUNNEL_INTERFACE_SET_OID 的对象标识符（OID）保留给系统使用。 不要在您的驱动程序中使用它。

## <a name="requirements"></a>要求

**版本**： Windows 7 和更高版本

**标头**： Ntddndis （包括 Ndis .h）
