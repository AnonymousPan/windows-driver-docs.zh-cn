有关将 NDIS 1.x 或更早版本的驱动程序移植到 NDIS 1.x 的信息，请参阅 [将 ndis 1.X 驱动程序移植到 ndis 6.0](/previous-versions/windows/hardware/network/porting-ndis-5-x-drivers-to-ndis-6-0)。
