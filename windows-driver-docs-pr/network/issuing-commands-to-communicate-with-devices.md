# <a name="issuing-commands-to-communicate-with-devices"></a>发出命令以便与设备通信





如果某个小型端口驱动程序具有一个用于控制总线上的设备的 WDM 下边缘，则会向总线接口发出命令，以便与这些设备通信。 有关详细信息，请参阅 [处理 irp](../kernel/handling-irps.md) 和特定总线驱动程序接口的文档。
