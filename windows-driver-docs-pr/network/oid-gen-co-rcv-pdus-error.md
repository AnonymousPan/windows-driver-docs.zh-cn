# <a name="oid_gen_co_rcv_pdus_error"></a>OID_GEN_CO_RCV_PDUS_ERROR

OID_GEN_CO_RCV_PDUS_ERROR OID 指定 NIC 接收到的 Pdu 数，但由于错误而未指示绑定协议。

## <a name="requirements"></a>要求

**版本**： Windows Vista 和更高版本的**标头**： Ntddndis （包括 Ndis .h）
