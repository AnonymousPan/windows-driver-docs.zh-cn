# <a name="remote-ndis-data-message"></a>远程 NDIS 数据消息





远程 NDIS 设备封装 NDIS 数据包，以在数据通道之间传输它们。 数据消息用于执行此操作，因为它们可以包含带外 (OOB) 数据或每个数据包的信息。
