# <a name="oid_gen_co_rcv_crc_error"></a>OID_GEN_CO_RCV_CRC_ERROR

OID_GEN_CO_RCV_CRC_ERROR OID 指定接收的具有循环冗余检查（CRC）错误的 Pdu 的数目。

## <a name="requirements"></a>要求

**版本**： Windows Vista 和更高版本的**标头**： Ntddndis （包括 Ndis .h）
