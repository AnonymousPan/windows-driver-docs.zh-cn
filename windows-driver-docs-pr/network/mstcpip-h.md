# <a name="mstcpiph"></a>Mstcpip.h

本部分包含 Mstcpip 标头的内核模式网络驱动程序主题。 此标头包含在 Windows SDK 中，因为它也与用户模式网络应用程序共享。

Mstcpip 标头包含对核心 Winsock 定义的 Microsoft 特定扩展的定义。
