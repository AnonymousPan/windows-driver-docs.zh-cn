# <a name="wdi_frame_id"></a>WDI_FRAME_ID

WDI_FRAME_ID 的数据类型是定义框架 ID 的 UINT16 值。 这只是一个标识符。 它不会传达有关帧顺序的信息。

```c++
typedef UINT16 WDI_FRAME_ID;
```

## <a name="requirements"></a>要求

**支持的最低客户端**： Windows 10

**支持的最低服务器**： Windows server 2016

**标头**： Dot11wdi
