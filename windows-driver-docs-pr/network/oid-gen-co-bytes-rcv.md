# <a name="oid_gen_co_bytes_rcv"></a>OID_GEN_CO_BYTES_RCV

OID_GEN_CO_BYTES_RCV OID 指定了未出现错误的 Pdu 中接收的字节数。

## <a name="requirements"></a>要求

**版本**： Windows Vista 和更高版本的**标头**： Ntddndis （包括 Ndis .h）
