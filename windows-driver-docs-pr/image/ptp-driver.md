# <a name="ptp-driver"></a>PTP 驱动程序

图片传输协议（PTP）驱动程序使 PTP 设备能够支持 WIA 驱动程序模型。

此驱动程序是使用 WIA 体系结构编写的，符合图片传输协议（PTP）标准。

有关这些结构的定义，请参阅[ISO 15740:2013 图片传输协议](https://www.iso.org/obp/ui/#iso:std:iso:15740:ed-3:v1:en)网站。
